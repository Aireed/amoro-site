# Roadmap

This roadmap displays the workings that the Amoro community plans to complete in 2024 and their priorities. Each item is associated with a Github issue, where you can learn about its specific design and latest developments.

# Priority 1

* [Refactor AMS using function and process api](https://github.com/apache/amoro/issues/2596)
* [Add metric collection for ams](https://github.com/apache/amoro/issues/1528)
* [Support table sorting process](https://github.com/apache/amoro/issues/1360)
* [Serverless process: automatic scaling for process resources](https://github.com/apache/amoro/issues/2531)
* [Monitor the health status of tables](https://github.com/apache/amoro/issues/2532)
* [Improve Mixed format](https://github.com/apache/amoro/issues/2533)

# Priority 2

* [Support Apache Hudi](https://github.com/apache/amoro/discussions/2258)
* [Support ingestion process](https://github.com/apache/amoro/discussions/2535)
* [Maintain secondary indexes on tables](https://github.com/apache/amoro/discussions/2536)
* [Collect scan metrics and improve optimizing rules based on them](https://github.com/apache/amoro/discussions/2537)
* [Enhance the management capabilities of the Paimon format](https://github.com/apache/amoro/discussions/2538)
